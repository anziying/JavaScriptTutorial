函数

函数可包含0或数个argument，有或无返回值（return; 语句可直接退出函数，无返回值）。

函数变量分为全局变量和局部变量，此处与各类高级语言类似，不多赘述。

变量

未声明的变量不可配置全局属性，已声明的变量可配置变量属性。

var var1 = 1;
var2 = 2;
delete var1; //false, 不可删除
delete var2; //true, 可删除

如果var 变量名 定义在函数作用域之外，则属于全局变量。 如果var 变量名 定义在函数作用域之内，则属于局部变量。 如果变量未定义且出现在函数作用域之内，则属于全局变量。但需要在函数调用后才能在全局看见，否则undefined。

字符串

字符串可以被定义成String，也可以被定义成Object。此处建议不要使用Object定义字符串，否则可能会造成一些麻烦。（例如使用绝对等于时Object类型不能等于String类型）

var x="John"; //x类型为String
var y= new String("John") //y类型为Object

运算符

JS的数学运算符与C类似，包括+，-，*，/，%，++，--，+=，-=，*=，/=，%=等。直接使用+可以拼接字符串。将字符串与数字相加，会得到字符串，无需自行转化。

JS的比较、逻辑运算符有==，===，!=，!==，>，<，>=，<=，&&，||，!等。功能与其他语言中运算符类似。

JS拥有三元运算符，下例：

异常处理

try-throw语句结构：

try {
//此处尝试一个操作，例如
if(x == 0) throw "x是0"
... //异常的抛出
} catch(e) {
//catch代码块只有异常时触发
//比如，此处可以有一个e.message的输出
... //异常的捕获与处理
} finally {
//无论前面异常与否，finally都会被执行
... //结束处理
}
